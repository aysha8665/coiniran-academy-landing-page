# Coiniran Academy Landing Page


### Install
```
npm install
```

Make sure you have [Hugo installed](https://gohugo.io/getting-started/installing/#homebrew-macos) as well.

### Usage
```
npm run start
```

open a browser and go to http://localhost:1313
